无偏估计

通常，我们会选择通过样本来估计整体的一些性质，比如通过样本均值估计整体的期望。分析一个估计量（比如这里的样本均值）好不好有很多指标，第一个要说的就是无偏性，就是我们如果有很多样本，然后我们计算这些样本的均值的均值，假如随着样本不断增多，这个均值的均值会趋向于整体的真实期望，就是无偏了，也就是说，没有偏差了。

除了无偏性，还会有一个有效性，同样也用这个样本均值作为例子，随着样本越来越多，每次样本均值的总体均值可能会越来越趋向真实期望，可是每次均值的分布是怎样呢，很明显，如果我们的这个样本均值分散比较集中在真实期望附近，肯定会比散布在较远的地方要好，这时候就涉及到方差的问题了，所以，我们可以通过样本均值的方差，去判断这个样本均值到底在这个情况下是不是一个好的估计量。

进一步说明，无偏性和有效性是不相关的，因为均值可以分布很集中，但集中在不是期望的地方，这时候这个期望就是有偏有效的，以此类推，什么情况组合都可以。

最后，强调一下，以上说明的是以样本均值作为估计量去估计整体的期望的例子，我们通过均值的均值和均值的方差去分析它的无偏性和有效性，为什么要这样强调，因为我们还可以通过样本的方差估计整体的方差，用方差的均值和方差的方差分析它的无偏性和有效性。

之所以先说样本均值再说样本方差，是因为正常的样本均值是无偏的，可是如果我们用我们印象中的方差共识计算样本的方差，它是有偏的，这个也就是重点问题了，为什么无偏估计的方差应该是n-1。

我们知道，方差的定义公式是：
$$\sigma ^2 = \frac{1}{n} [\sum _{i-1} ^n (x_i - \mu)^2]$$
我们希望估计的是整体的方差，那么我们就需要用到整体的期望，可是我们不知道，只知道样本的期望，所以，我们就先尝试用样本的期望代进去看有什么效果：
$$\sigma ^2 = \frac{1}{n} [\sum _{i-1} ^n (x_i - \overline x)^2]$$
$$\sigma ^2 = \frac{1}{n} [\sum _{i-1} ^n (x_i^2 - 2\overline x x_i + \overline x ^2)]$$
$$E(\sum _{i=1}^n (x_i - \overline x)^2) = \sum _{i=1}^n E(x_i ^2) - nE(\overline x ^2)$$
还记得方差还等于用均值平方减去平方均值，所以平方的均值可表示为：
$$E(x ^2) = Var(x) + (E(x))^2 = \sigma ^2 + \mu ^2$$
$$E(\overline x ^2) = Var(\overline x) + (E(\overline x))^2 = \frac{1}{n} \sigma ^2 + \mu ^2$$
把这两个式子代回原式得到：
$$E(\sum _{i=1}^n (x_i - \overline x)^2) = \sum _{i=1}^n (\sigma ^2 + \mu ^2) - n*(\frac{1}{n}\sigma ^2 + \mu ^2) = (n-1)\sigma ^2$$
总的来说，就是我们并不知道整体的期望，直知道一些样本的均值，所以我们用样本的均值去估计整体的方差的时候，就会导致一些偏差（也就是上述推导的结果），从推到的结果可以看出，如果想要修正这个偏差，让方差的估计变成无偏估计，就需要把原来的方差公式除以一个n-1
