三大抽样分布

很多统计推断都是基于正态分布的假设，以服从标准正态分布的随机变量为构造的三种统计量所服从的分布就是三大抽样分布：卡方分布、t分布、F分布

卡方分布的定义是：设随机变量xi独立同分布，服从标准正态分布，令:
$$X=\sum_{i=1}^n x_i^2$$
则称X是自由度为n的卡方变量，其分布称为自由度的n的卡方分布。也可以说，n个独立的标准正态分布变量的平方和服从自由度为n的卡方分布。自由度n越大，卡方分布的密度曲线越趋于对称，当n很大时，卡方分布近似为正态分布。

卡方分布一般用于总体方差未知时通过样本方差对总体方差的估计，比如求置信区间。用一个具体例子说明一下，比如现在我们分析了101个样本得到样本方差为0.56，分析方差的90%置信区间，首先构建卡方统计量：
$$X^2 = \frac{(n-1)S^2}{\sigma ^2}$$
需要说明的是，这个卡方统计量和上面定义的卡方变量虽然形式不同，卡方变量可以理解成更通用的表达，这里的卡方统计量则主要用于区间估计。得到卡方统计量后，有：
$$P(x_1 < \frac{(n-1)S^2}{\sigma ^2} < x_2) = 0.9$$
根据自由度101我们可以确定卡方分布的具体形式，确定x1、x2使得概率为0.9，然后再把样本方差代进去就能得到总体方差的90%置信区间了，基本上区间估计都是这个思路的了。

第二种分布是t分布，主要用于根据小样本来估计成正态分布且方差未知的总体的均值。假设X是呈正态分布的独立随机变量，期望为mu，方差为sigma^2，令样本均值和样本方差：
$$\overline X_n = \frac{X_1+...+X_n}{n}$$
$$S_n^2 = \frac{1}{n-1} \sum_{n=1}^n (X_i-\overline X_n)^2$$
$$T=\frac{\overline X_n - \mu}{S_n/ \sqrt{n}}$$
为自由度为n的t变量，其分布称为自由度为n的t分布。

t分布和标准正态分布的概率密度函数图像很接近，都是以原点对称，单峰偶函数，在x=0取到最大值，但t分布的峰值低于标准正态分布的峰值。

t分布在区间估计中会用到，比如在方差未知的情况下，我们为了估计期望，会构建一个pivotal quantities（其实就是标准化处理的均值）：
$$Q=\frac{\sqrt n (\overline x - \theta)}{S}$$
$$S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \overline x )^2$$
简单来说，就是专门针对方差未知的总体均值研究得到的分布情况，就是t分布。顺带一提，如果方差已知，那么标准化后的均值服从的是标准正态分布而不是t分布。同时也有另一个想要说明的，是之所以强调pivotal quantities而不是原始的期望，是经过处理后得到的Q，它服从的分布和theta是无关的，所以我们才能在theta未知的情况下，仍然能得到Q服从的标准正态分布（方差已知）或者t分布（方差未知）。

最后再简单介绍一下t分布在区间估计中的具体例子，假如现在要估计一个总体方差未知的总体期望的95%置信区间，我们就可以先计算出n个样本的期望和方差，代进T里面，得到自由度为n的t分布，然后我们就能确定出累计概率为2.5的点，比如t1，因为t分布关于x=0堆成，同样得到t2，这样，我们就能知道：
$$P(t1 < T < t2)=0.95$$
把T完整形式写进去，转换一下就得到总体期望的95%置信区间了

F分布的定义是，设随机变量X、Y服从自由度为m和n的卡方分布，XY相互独立，则称
$$F=\frac{X/m }{Y/n}$$
为自由度为m和n的F变量，其分布称为自由度为m和n的F分布。

F分布具有以下性质：（1）若Z～F_m,n，则1/Z~F_n,m (2)若T~t_n，则T^2~F_1,n (3)F_m,n (1-a)=1/F_n,m (a)
