假设检验实例

之前介绍了假设检验的基本思想，现在再介绍一些具体的假设检验方法。

第一个是Z检验，它是用于大样本（样本容量大于30）的均值的差异性检验的方法，换句话说就是检查两个平均值之间是否具有显著的差异，这两个平均值一般是样本的均值和总体的期望，当然也可以比较两个样本的均值之间的差异。

举一个例子来结合说明，假设我们现在要做假设检验，原假设为样本期望和总体期望没有显著差异，备择假设是有差异。为了进一步进行验证，我们可以构建统计量Z值：
$$Z = \frac{\overline X - \mu_0}{\sigma / \sqrt n}$$
其中，总体期望为mu0，样本的均值为x杠，sigma是总体方差，n是样本容量。

为什么我们要这样构建z统计量，为什么它叫z统计量，说到底还是因为中心极限定理：大量独立同分布的随机变量的均值经过标准化后，依概率收敛到标准正态分布。

实质上，经过构建的统计量，Z就会服从标准正态分布。我们可以想象一下，样本期望最完美的情况，应该是样本期望的等于总体期望，这时候z就等于0，也就是位于正态分布的中心，也就是期望的位置，这个概率也是最大的，如果样本期望偏离了（样本期望和总体期望之间有差异），这就是我们希望分析的问题。样本期望当然不可能完美等于总体期望，那么偏离的程度怎样，这时候就可以定义显著性水平了，一般来说我们会定义显著性水平为0.05，也就是z所服从的正态分布中，概率为0.05的区域，也就是所谓的拒绝域，也就是样本期望和总体期望相差较大的区域。所以，我们通过定义了拒绝域，再观察统计量z有没有落在拒绝域，就能判断在显著性水平为0.05的情况下，样本期望和总体期望之间有没有显著差异了。

最后再补充一点，一开始我们要求z检验的样本量要足够大，是因为我们在构建z值的时候用到了中心极限定理，它当然需要样本量足够大才能成立。

还有一点，我想说明一下p值在z检验中应用在哪里，其实我们在构建了z统计量之后，有两种思路去解决z检验。第一种是根据求出来的z值在正态分布的哪个点，计算正态分布小于那个点的概率，再乘以二得到p值（这里是p值的定义，考虑比样本更极端的情况，因为标准正态分布是对称的所以乘以二），对比p值和0.05的大小。第二种方法就是求出显著性水平为0.05在标准正态分布对应的区域，也就是负无穷到-1.95，1.95到正无穷，这里就是拒绝域，分析z值有没有落在这个区域。

以上就是z检验的基本思路和过程。

第二个是t检验，上面的z检验只在样本容量大的时候才有效，容量小具有一定误差，可能转换的z值就不服从正态分布了，所以对于小样本的情况，我们需要t检验去检测平均值之间的差异。顺带一提，z检验需要知道总体方差，t检验不需要。

基本过程和Z检验类似，主要是统计量我们换成了t统计量：
$$T = \frac{\overline X - \mu_0}{S/ \sqrt {n}}$$
$$S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \overline X)^2$$
总的来说和z检验唯一的区别就是在构建统计量的时候，因为总体方差未知，所以用总体方差的无偏估计代替了它，注意是无偏估计，所以S^2才会除以n-1。

简单总结一下，如果我们想要分析样本期望是否于总体期望具有显著差异，可以进行假设检验，根据情况选择z检验或者t检验，如果样本量小就选择t检验，如果总体方差未知也选择t检验，事实上，哪怕样本量大，选择t检验效果也挺好的，所以选择t检验总是对的，当然做题的时候如果给出了总体方差，出于解题的角度还是选择z检验比较好，不然就要手算样本方差了。

前面两个是分析均值之间的差异的，自然也会想到有没有检验样本方差和总体方差的差异，答案当然是有的，这个检验叫F检验。

F检验的过程还是和上面的差不多，首先构造统计量：
$$F = \frac{S^2}{\sigma_0^2 / (n-1)}$$
$$S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \overline X )^2$$
现在要探讨的主要问题是为什么需要构建这样一个统计量，他有什么作用，我觉得主要就是我们现在分析样本方差和总体方差的差异，这种差异可以通过两者的差或者两者的商去判断，那么为什么我们构建的统计量F不直接采用两个方差的商，还要乘上一个（n-1）呢，这里就是重点了！因为，我们不知道两者的商服从什么分布，但是！我们来看看服从卡方分布的变量的形式：
$$X = \sum_{i=1}^k Z_i^2$$
再把F检验的统计量转化一下：
$$F = \frac{S^2}{\sigma_0^2 / (n-1)} = \frac{\sum_{i=1}^n (X_i - \overline X )^2 / (n-1)}{\sigma_0^2 / (n-1)} = \sum_{i=1}^n \frac{(X_i - \overline X )^2}{\sigma_0^2}$$
到这里就很清楚了，事实上，我们做的就是想办法让样本方差和总体方差的商服从卡方分布，之所以分母除以一个(n-1)就是为了抵消分子那部分，因为这些都是常数，所以对分析总体方差和样本方差之间的差异没有影响。

最后总结一下，不论是z检验、t检验还是f检验，说到底，他们就是想分析均值之间的差异或者方差之间的差异，分析差异我们可以通过商或者差分析，可是得到了商或者差之后怎么搞出一个通用的方法，去判断这种差异是否显著，那就是把他们通过适当的方式转化成我们已知的服从一定分布的变量，比如正态分布，比如卡方分布，通过这些分布，我们可以知道差异的分布的具体情况，然后我们就可以通过确定显著性水平，确定差异比较小概率出现的状况，并认为如果这种差异出现了，就可以认为均值或者方差之间的差异足够大了。所以说，假设检验，说到底最关键的还是根据不同情况，选择统计量。
