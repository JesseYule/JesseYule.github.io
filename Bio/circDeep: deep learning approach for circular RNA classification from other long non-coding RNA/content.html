Deep learning of the back-splicing code for circular RNA formation

感觉现在写的这些生物信息的内容越来越偏门了，但毕竟我也不想做流量博主，还是希望能多分享一些别人少写的内容，帮助有需要的人。另外我也不是学生物的，所以或许会有一些知识性错误，欢迎指出。

这次阅读的论文主要研究的是circRNA的反向剪接，事实上，circRNA就是RNA（一般指pre-mRNA）在剪接时，外显子的50和30末端通过反向剪接共价连接形成circRNA的
并非所有外显子都能参与反向剪接，有多种因素会影响到circRNA的形成，或者简单来说，就是你不能说一个片段头尾能碱基互补配对就能形成circRNA，可能还会受到其他片段的调控，所以我们才需要深度学习进行分析，判断哪些片段能进行反向剪接，哪些不能。
数据方面，主要是从CircRNADB和CircBase中获取外显子circRNA，对于每个circRNA，都有两个剪接位点的基因组位置，从两个剪接位点各提取前后50nt的序列得到总长度为200的序列作为positive数据，对于negative数据，则是选择相同转录本中但未包括在positive数据中的另一对剪接位点进行数据的提取，这样，我们就得到了长度为200的输入序列了，然后再对AUCG进行独热编码，得到模型的输入。

其实从模型的结构图就可以看出，整个模型挺简单的，仅仅只是用了两个卷积层，加上最大池化，就展开用全连接层做分类了。

虽然最后做出来的效果好像比前人好，但确实没什么值得细说的。
